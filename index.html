Visuel & Accessibilité

Optimisez la visibilité pour les étudiant(e)s. Augmentez la taille des éléments, utilisez des indicateurs visuels et facilitez le suivi du curseur pour une meilleure expérience d'apprentissage.

🖱️

Pointeur de souris

Choisissez un pointeur de couleur vive et plus gros pour une meilleure visibilité sur grand écran. Essentiel pour que les étudiant(e)s suivent les actions.

Curseur de texte

Personnalisez la couleur et la taille du curseur de texte pour une meilleure visibilité lors de la saisie.

🔍

Loupe Windows

Win + (plus)

Agrandissez instantanément n'importe quelle partie de l'écran. Win + (plus) pour agrandir, Win + (moins) pour réduire, Win + Échap pour quitter.

✨

Clics visuels PowerToys PowerToys

Win + MAJ + H

Affiche des cercles colorés lors des clics (jaune = gauche, bleu = droit). Les étudiant(e)s voient exactement où les clics se produisent.

⭕

Localisateur souris PowerToys

Ctrl + Ctrl

Appuyez 2 fois sur Ctrl pour afficher un cercle blanc autour du curseur. Idéal pour retrouver la souris rapidement sur grand écran.

🎨

Thème clair

Basculez en mode clair pour une meilleure lisibilité lors de la projection. Le contraste est supérieur sur projecteur.

🔤

Taille du texte

Augmentez la taille du texte système pour améliorer la lisibilité à distance. Recommandé : 125 % ou 150 %.

🖱️

Souris sans-fil pour diaporama

Utilisez une souris sans-fil pour changer de diapositive. Astuce : Dans PowerPoint, désactivez le menu contextuel (clic droit) pour utiliser le bouton droit comme retour arrière.

Vie Privée & Confidentialité

Protégez les informations personnelles et évitez les distractions pendant les cours. Ces outils aident à présenter du contenu sans exposer la navigation privée, les favoris personnels ou les notifications.

⭐

Edge : Barre de favoris

Ctrl + MAJ + B

Masquez la barre de favoris pour éviter d'exposer des sites personnels. Le raccourci fonctionne instantanément dans Edge.

👤

Edge : Profils séparés

Créez un profil "Travail" et un profil "Personnel". Les favoris, historique, extensions et mots de passe restent complètement isolés entre les profils.

Guide sur profils Edge
🔕

Ne Pas Déranger

Win + N

Activez le mode Focus pour bloquer les notifications et éviter les interruptions pendant vos présentations.

🖥️

Icônes du Bureau

Masquez les icônes du bureau pour une présentation plus propre et éviter d'exposer des fichiers personnels.

Aperçu des messages Teams

Par défaut, Teams affiche un aperçu du message en bas à droite de l'écran. Si un collègue vous envoie un message d'ordre privé pendant que vous projetez, tout le monde le verra. Désactivez les aperçus dans les paramètres Teams.

Aperçu des courriels Outlook

Outlook affiche lui aussi un aperçu des courriels entrants en bas à droite de l'écran — objet et début du message inclus. Désactivez-le pour éviter qu'un courriel délicat s'affiche devant la classe.

🔇

Notifications sonores

Un « ding » de Teams ou d'Outlook en plein exposé brise la concentration de tout le monde. Coupez le son des notifications indépendamment du volume général pour garder le contrôle.

🔎

Historique dans la barre d'adresse

Dès que vous commencez à taper dans la barre d'adresse d'Edge, le navigateur suggère des sites visités récemment. Supprimez l'historique avant le cours ou utilisez un profil dédié à l'enseignement.

🔍

Suggestions de recherche Google / Bing

Les moteurs de recherche mémorisent vos recherches passées et les proposent en autocomplétion. Effacez l'historique de recherche ou utilisez une fenêtre de navigation privée (Ctrl + MAJ + N) pour les démonstrations — aucune suggestion personnelle ne s'affiche.

🗂️

Bureaux virtuels

Win + Tab

Créez un bureau dédié à l'enseignement, séparé de vos espaces personnels. Naviguez entre bureaux avec Win + Ctrl + ← ou Win + Ctrl + →.

Guide sur Bureaux virtuels

Edge : Collections

Ctrl + MAJ + Y

Organisez les ressources pédagogiques dans des collections privées. Parfait pour préparer les cours sans mélanger avec la navigation personnelle.

Guide sur Collections

Les espaces de travail permettent de partager un groupe d'onglets en temps réel avec d'autres personnes. Idéal pour co-naviguer avec les étudiant(e)s : tout le monde voit les mêmes pages en même temps, sans avoir à envoyer de liens.

Configuration Présentation

Gérez efficacement les écrans et l'espace de travail. Connectez-vous aux projecteurs, organisez les fenêtres et optimisez l'affichage pour l'enseignement.

📽️

Contrôle projecteur

Extinction image (AV-Mute) : Rend l'affichage noir sans débrancher le projecteur.
Pause image (Freeze) : Fige l'image projetée pour travailler sur le portable en privé.

📺

Mode projection

Win + P

Choisissez rapidement entre Dupliquer (même affichage) ou Étendre (bureau étendu). Essentiel pour gérer projecteur et notes.

📡

Connexion sans-fil

Win + K

Connectez-vous rapidement à une télévision compatible. Plus besoin de chercher dans les paramètres.
Note : Fonctionne avec certaines télévisions intelligentes. Non compatible avec les projecteurs traditionnels.

🪟

Ancrage fenêtres

Win + ←/→

Ancrez les fenêtres rapidement. Win + Flèche pour placer à gauche/droite. Idéal pour comparer deux documents côte à côte.

Guide ancrage
🌐

Mode plein écran

F11

Passez en plein écran dans le navigateur. Maximise l'espace pour le contenu et élimine les distractions.

⏱️

Chronomètre pour gestion du temps

Free Countdown Timer : application portable, toujours visible à l'écran, personnalisable et en français. Idéale pour gérer le temps des activités et exercices en classe.

🖐️ Gestes du pavé tactile

Sur le portable, plusieurs personnes privilégient la souris externe par habitude. Pourtant, quand on maîtrise bien les gestes du pavé tactile, c'est souvent plus rapide en classe et on évite de transporter une souris pour rien.

🧭 Navigation et lecture

✌️ 2 doigts
Défilement vertical et horizontal dans les pages, PDF et documents. Idéal pour parcourir un document projeté sans perdre le rythme.
🤏 Pincer / écarter
Zoomer ou dézoomer pour adapter la lisibilité en direct selon la classe.
✌️ 2 doigts (cliquer)
Clic droit rapide pour ouvrir les menus contextuels.

🪟 Fenêtres et multitâche

🖖 3 doigts vers le haut
Ouvrir la vue des tâches pour retrouver vite une application.
🖖 3 doigts gauche / droite
Basculer rapidement entre les fenêtres ouvertes (présentation, navigateur, Teams).
🖖 3 doigts vers le bas
Afficher le bureau pour masquer toutes les fenêtres d'un coup.
🖐️ 4 doigts vers le haut
Afficher les bureaux ouverts et choisir rapidement le bon espace de travail.

🗂️ Gestion des bureaux virtuels

🖐️ 4 doigts gauche / droite
Passer d'un bureau virtuel à l'autre (ex. bureau cours / bureau personnel).

🔎 Actions rapides Windows

🖖 3 doigts (cliquer)
Ouvrir la recherche Windows pour lancer une application ou trouver un fichier rapidement.
🖐️ 4 doigts (cliquer)
Ouvrir le centre de notifications et le calendrier; pratique pour activer rapidement Ne pas déranger avant un cours.

Astuce : personnalisez ces gestes dans Paramètres > Bluetooth et périphériques > Pavé tactile pour les adapter à vos habitudes. Ouvrir le pavé tactile

⚡ Préparation de cours recommandée

1

Avant le cours

Activez le mode clair, masquez icônes du bureau, pointeur de couleur vive, mode Ne pas déranger (Win + N), masquez favoris Edge (Ctrl + MAJ + B).

2

Connexion projecteur

Appuyez Win + P pour choisir le mode d'affichage. "Dupliquer" sans notes, "Étendre" pour garder des notes privées sur l'écran du portable.

3

Pendant le cours

Utilisez Win + (plus) pour agrandir, Ctrl + Ctrl pour localiser souris, Win + MAJ + H pour clics visuels (PowerToys), F11 pour plein écran.

4

Après le cours

Désactivez mode Ne pas déranger, réactivez favoris Edge si désiré, revenez en mode sombre selon la préférence.
